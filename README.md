# Backend with Frontend Integration

## 🚀 Quick Start

```bash
npm install
npm start
```

Then open: **http://localhost:3000**

## 📁 Files

- `server.js` - Express backend with API routes
- `index.html` - Frontend HTML (served by Express)
- `style.css` - Frontend CSS
- `script.js` - Frontend JavaScript with API integration
- `package.json` - Dependencies

## ✅ Features

- JWT authentication
- Bcrypt password hashing
- Role-based access control
- CRUD for Accounts, Departments, Employees, Requests
- API + Frontend in one server
